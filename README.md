# Библиотека промптов (Prompt Library)

Библиотека структурированных промптов для различных задач с метаданными, примерами и тест-кейсами.

## Структура

```
prompts/
  └── recipe_json_v1.yaml    # Промпт для генерации JSON-рецептов
```

## Формат карточки промпта

Каждый промпт представлен в виде YAML-файла со следующей структурой:

- **name** — название промпта
- **version** — версия промпта
- **owner** — владелец/автор
- **tags** — теги для категоризации
- **goal** — краткое описание цели промпта
- **prompt** — текст промпта с переменными
- **examples** — примеры использования (вход/выход)
- **test_cases** — тест-кейсы для валидации

## Промпты

### Генератор структурированного JSON-рецепта (v1.0)

**Теги:** `recipes`, `json`, `nutrition`, `image_prompt`, `ru`, `food`

**Цель:** Генерация реалистичного рецепта блюда в виде валидного JSON с описанием, пошаговой инструкцией, нутриционной ценностью и промптом для генерации изображения.

**Входные переменные:**
- `{dish_name}` — название блюда (опционально)
- `{ingredients}` — список ингредиентов (опционально)
- `{cuisine}` — тип кухни (опционально)

**Файл:** `prompts/recipe_json_v1.yaml`

## Использование

1. Выберите нужный промпт из папки `prompts/`
2. Замените переменные в тексте промпта на реальные значения
3. Отправьте промпт в LLM (GPT, Claude и т.д.)
4. Получите структурированный ответ в формате JSON

## Вклад в библиотеку

При добавлении нового промпта:
1. Создайте YAML-файл в папке `prompts/`
2. Следуйте стандартной структуре карточки промпта
3. Добавьте описание в этот README
4. Создайте Pull Request
